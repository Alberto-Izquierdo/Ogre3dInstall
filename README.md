# Ogre3D Install

Scripts to install Ogre3d (1.9), CEGUI (0.8), CEED (0.8) and OgreBullet on Debian-based linux distributions.

Tested on Debian and Ubuntu.


### How to install

Run the scripts in order (install-ogre.sh before install-ogre-bullet.sh).
